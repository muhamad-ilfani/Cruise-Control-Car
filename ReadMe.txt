Radar_Acc_ExtSen_CS9 untuk metode switching
Radar_Acc_ExtSen_CS9_23Apr untuk metode constraint

Cara menjalankan code
1. Buka carsim, masukan simulink ke dalam carsim
2. Buka simulink dari carsim (send to simulink)
3. Buka msinitctrl_20Feb untuk metode switching, msinitctrl_23Apr untuk metode constraint melalui matlab
4. Run untuk mendapatkan modelnya
5. Run melalui carsim
